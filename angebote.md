## Paket 1 – Starter (600 €)

**Für wen:** Freiberufler die einfach online sichtbar sein wollen – ohne Schnickschnack.

**Leistungen:**
- 3 Seiten: Startseite, Über mich, Kontakt
- Professionelles Design (an bestehendes Branding angepasst)
- Kontaktformular (kein eigener Server nötig)
- Mobil-optimiert (Smartphone, Tablet)
- Basis-SEO (Titel, Beschreibungen, Google-tauglich)
- Einrichtung auf GitHub Pages (kostenlos, zuverlässig)
- Eigene Domain verbinden (z.B. max-mustermann.de)
- Übergabe als fertige, funktionierende Website

**Nicht enthalten:**
- Texte schreiben (Kunde liefert Rohtexte)
- Logo-Erstellung
- Bildbearbeitung / Fotoshooting
- Nachträgliche Änderungen

**Lieferzeit:** 1–2 Wochen nach vollständigem Briefing

**Nicht enthalten:**

- Bildbearbeitung / Fotoshooting
- Laufender Support nach Übergabe

**Lieferzeit:** 2–3 Wochen nach vollständigem Briefing
